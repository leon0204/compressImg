# compressImg
批量异步压缩图片


#### 分开处理 jpg 和 png 
- jpg 采用 python 的 PIL 库 处理。
- png 采用 tinyapi 处理




#### 使用方法
进入项目根目录
```
python tinypng.py -r <源目录> <目标目录>

```

如果不填目标目录，默认为覆盖原图
